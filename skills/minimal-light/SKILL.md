# Minimal Light Design System

When creating or styling PowerPoint slides, follow these guidelines for an ultra-clean, minimalist aesthetic.

## Color Palette

| Role            | Hex      | Usage                              |
|-----------------|----------|------------------------------------|
| Background      | FFFFFF   | Slide backgrounds                  |
| Surface         | FAFAFA   | Card backgrounds, subtle sections  |
| Primary Text    | 1A1A1A   | Headings, body text                |
| Secondary Text  | 6B7280   | Captions, dates, metadata          |
| Accent          | 2563EB   | Key highlights, links, one accent  |
| Accent Light    | DBEAFE   | Subtle highlight backgrounds       |
| Border          | E5E7EB   | Dividers, table borders            |
| Success         | 059669   | Positive indicators                |
| Warning         | D97706   | Caution indicators                 |
| Error           | DC2626   | Negative indicators                |

## Typography

- **Title slides:** fontSize 44–52, bold, color `1A1A1A`
- **Section headers:** fontSize 32–36, bold, color `1A1A1A`
- **Slide headings:** fontSize 24–28, bold, color `1A1A1A`
- **Body text:** fontSize 16–18, color `1A1A1A`, line spacing 1.4
- **Captions:** fontSize 12–14, color `6B7280`
- **Font family:** Inter, Helvetica Neue, or Arial. No serifs.

## Layout Principles

- **White space is the design.** Leave large margins and gaps between elements.
- **Format:** Always use 16:9 format with 10x5.625 inches
- **Title position:** x: 1.0, y: 0.8, w: 8, h: 0.9
- **Subtitle position:** x: 1.0, y: 1.7, w: 8, h: 0.5
- **Body content area:** x: 1.0, y: 2.5, w: 8, h: 4.0
- **Inset margins:** At least 1.0" from left and right edges for a centered, editorial feel.
- **Single column preferred.** Use two columns only when comparing items side by side.

## Shapes & Accents

- Minimal use of shapes. Prefer a single thin horizontal line (h: 0.02, fill `E5E7EB`) as a divider.
- If using callout boxes: fill `FAFAFA`, border color `E5E7EB`, no rounded corners.
- One accent color only (`2563EB`). Use it for a single keyword, number, or icon per slide.

## Tables

- Header row: fill `FAFAFA`, color `1A1A1A`, bold, fontSize 14
- Body rows: fill `FFFFFF`, color `1A1A1A`, fontSize 13
- Border: pt 1, color `E5E7EB`
- No alternating row colors — keep it uniform.

## Charts

- Use monochrome tones: `2563EB`, `93C5FD`, `DBEAFE`, `6B7280`, `D1D5DB`
- Minimal gridlines, no background fill
- Data labels only when essential

## Do's and Don'ts

- ✅ Embrace emptiness — less is more
- ✅ One key message per slide
- ✅ Use large font sizes and few words
- ✅ Align everything to a clean grid
- ❌ Don't use more than one accent color
- ❌ Don't use gradients, shadows, or 3D effects
- ❌ Don't fill slides with content — leave at least 40% white space
- ❌ Don't use bullet points if you can use a single sentence instead
