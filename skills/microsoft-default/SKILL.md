# Microsoft Default Design System

When creating or styling PowerPoint slides, follow these design guidelines for a clean, professional Microsoft-style look.

## Color Palette

| Role            | Hex      | Usage                              |
|-----------------|----------|------------------------------------|
| Primary         | 0078D4   | Headings, accent shapes, links     |
| Secondary       | 005A9E   | Sub-headings, darker accents       |
| Dark Text       | 252423   | Body text, primary content         |
| Subtle Text     | 605E5C   | Captions, secondary info           |
| Light Background| F3F2F1   | Content area backgrounds           |
| White           | FFFFFF   | Slide backgrounds                  |
| Accent Green    | 107C10   | Success, positive indicators       |
| Accent Red      | D13438   | Warnings, negative indicators      |
| Accent Yellow   | FFB900   | Highlights, callouts               |

## Typography

- **Title slides:** fontSize 36–44, bold, color `252423`
- **Section headers:** fontSize 28–32, bold, color `0078D4`
- **Slide headings:** fontSize 24–28, bold, color `252423`
- **Body text:** fontSize 16–18, color `252423`
- **Captions / footnotes:** fontSize 12–14, color `605E5C`
- **Font family:** Use Segoe UI where supported; fall back to Calibri or Arial.

## Layout Principles

- **Margins:** Keep at least 0.5" padding from all slide edges.
- **Format:** Always use 16:9 format with 10x5.625 inches
- **Title position:** x: 0.5, y: 0.3, w: 9, h: 0.8
- **Subtitle position:** x: 0.5, y: 1.2, w: 9, h: 0.6
- **Body content area:** x: 0.5, y: 1.8, w: 9, h: 4.5
- **Two-column layout:** Left column w: 4.2, right column x: 5.3, w: 4.2
- **Slide dimensions:** 10" × 7.5" (standard widescreen 13.33" × 7.5" also accepted)
- **Consistent alignment:** Left-align body text. Center titles only on title slides.

## Shapes & Accents

- Use a thin horizontal rule (rect shape, h: 0.03, fill `0078D4`) below headings to separate sections.
- Rounded rectangles (rectRound) for callout boxes with fill `F3F2F1` and border color `D2D0CE`.
- Icon-style accent circles with fill `0078D4` and white text/numbers for numbered lists.

## Tables

- Header row: fill `0078D4`, color `FFFFFF`, bold, fontSize 14
- Body rows: alternating fills `FFFFFF` and `F3F2F1`, color `252423`, fontSize 13
- Border: pt 1, color `E1DFDD`
- Cell padding: Use margin options where available

## Charts

- Use the primary color sequence: `0078D4`, `005A9E`, `107C10`, `FFB900`, `D13438`, `605E5C`
- Include data labels where practical
- Title charts with fontSize 16, bold, color `252423`

## Do's and Don'ts

- ✅ Use generous white space — don't crowd slides
- ✅ Limit text to 6 bullet points per slide, max ~8 words each
- ✅ Use high-contrast text (dark on light, white on primary blue)
- ❌ Don't use more than 3 colors per slide
- ❌ Don't use font sizes below 12pt
- ❌ Don't center-align body text in content slides
